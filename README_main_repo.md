# ArtHOI: Taming Foundation Models for Monocular 4D Reconstruction of Hand-Articulated-Object Interactions

Official repository for ArtHOI.

## Overview

ArtHOI targets monocular 4D reconstruction of hand-articulated-object interactions, without requiring object pre-scans or multi-view capture.

For qualitative results and method figures, please visit:
- https://arthoi-reconstruction.github.io/
